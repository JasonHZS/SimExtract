# Experiments Package

This directory is reserved for the experiment framework that will enable
systematic comparison of the 4 attribution methods.

## Planned Features

- **ExperimentRunner**: Orchestrate experiments across multiple methods
- **Evaluator**: Evaluate and score attribution results
- **Comparator**: Generate side-by-side comparisons
- **Metrics**: Evaluation metrics (coverage, precision@k, etc.)
